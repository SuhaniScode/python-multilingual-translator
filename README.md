# Python Multilingual Text Translator (googletrans)
A clean, beginner-friendly multilingual text translation tool built in Python using the `googletrans` library.  
Translate text between **dozens of languages** instantly with a simple Jupyter Notebook interface.
This project is perfect for students, NLP beginners, and anyone learning how translation APIs work.
